# IoT Greenhouse Project

## Description

This is an IoT greenhouse project that allows monitoring and controlling of temperature, humidity, and soil moisture levels. The project includes a website that displays real-time data and charts of temperature, humidity, and soil moisture, as well as allowing users to control fan, light, and pump operations, change modes, and set ranges for automatic control of light, pump, and fan.

## Technologies Used

-   [Node.js](https://nodejs.org/)
-   [Express.js](https://expressjs.com/)
-   [MongoDB](https://www.mongodb.com/)
-   [MQTT.js](https://github.com/mqttjs/MQTT.js)
-   [Adafruit IO](https://io.adafruit.com/)
